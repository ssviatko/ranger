Detailed Description

Block of 8 tokens awaiting write to output stream.

Used by the encoder, here is where it stores tokens on their way to output in order to write out all 8 flags in one single byte rather than writing out single bits. Using whole bytes saves a lot of time over using the cbit library to output individual bits.
